Pengenalan FSTM Pengenalan FSTM Selamat datang ke FSTM

FSTM ditubuhkan pada 1999 adalah fakulti teknologi maklumat yang ditubuhkan untuk mengendalikan program teknologi maklumat di KUIS.


Kini, FSTM mempunyai dua buah jabatan utama iaitu Jabatan Teknologi Maklumat (Multimedia) dan Jabatan Sains Komputer.


Pelbagai program IT dipacukan bersama rakan teknologi industri seperti Microsoft, Redhat Academy, Cisco, Adobe dan Amazon Web Services.
